## 4. 프리미엄 비주얼 시스템 (v4.0 전면 신규)

> 감정 여정을 **시각 언어**로 번역하는 핵심 장. 모든 페이지·컴포넌트에 일관 적용한다.

### 4-1. 모션 언어 — 시네마틱 우아함

**v4.0 공통 easing: Quartic Out**

v3.0까지의 `easeOut`을 모든 애니메이션에서 **Quartic Out** 커브로 교체한다. 빠르게 시작해서 아주 천천히 멈추는 이 곡선은 "빠르지도 느리지도 않은 우아한 움직임"을 만든다.

```typescript
// lib/motion.ts — 사이트 전체 모션 상수
export const SARAJOO_EASE = [0.16, 1, 0.3, 1]; // Quartic Out
export const SARAJOO_DURATION = 0.8;

export const fadeInUp = {
  initial: { opacity: 0, y: 20 },
  whileInView: { opacity: 1, y: 0 },
  viewport: { once: true },
  transition: { duration: SARAJOO_DURATION, ease: SARAJOO_EASE },
};
```

**모션 유형별 상세:**

| 모션 | 적용 대상 | v3.0 | v4.0 |
|------|----------|------|------|
| **섹션 등장** | 모든 블록 | y:20, easeOut, 0.8s | y:30, **Quartic Out**, 0.8s |
| **텍스트 마스크** | 헤드라인 | 단순 fade | **clip-path 슬라이드** — 아래→위로 글자가 잘려 나타남 |
| **패럴랙스** | Hero 배경, 스토리 이미지 | 없음 | **이미지가 텍스트보다 0.3배 느리게** 스크롤 |
| **이미지 호버** | 갤러리, 제품 | scale:1.03 | scale:1.05 + **brightness:1.05** + Quartic Out 0.4s |
| **CTA 호버** | 구매 버튼 | scale:1.05 + 색상 | scale:1.05 + 색상 + **Quartic Out 0.3s** |
| **페이지 전환** | 모든 페이지 | opacity fade 0.5s | opacity fade **Quartic Out 0.6s** |
| **HOT/COOL 탭** | 탭 전환 | 배경 0.6s | 배경 **Quartic Out 0.8s** |
| **숫자 카운트** | Social Proof | 카운트업 | 카운트업 + **원형 프로그레스 링** |
| **갤러리 스트립** | 홈 | 수동 스크롤 | **자동 슬로우 스크롤(20px/s)** + 수동 드래그 |
| **FAQ 아코디언** | 제품 | AnimatePresence | AnimatePresence + **height auto** + Quartic Out |

### 4-2. 텍스트 마스크 애니메이션 (v4.0 신규)

> 헤드라인이 단순히 나타나지 않고, **아래에서 위로 잘려 나타나는** 시네마틱 효과

```typescript
// components/ui/TextReveal.tsx
"use client";
import { motion } from 'framer-motion';
import { SARAJOO_EASE } from '@/lib/motion';

export default function TextReveal({
  children,
  delay = 0,
  className,
}: {
  children: React.ReactNode;
  delay?: number;
  className?: string;
}) {
  return (
    <div className="overflow-hidden">
      <motion.div
        initial={{ y: '100%' }}
        whileInView={{ y: '0%' }}
        viewport={{ once: true }}
        transition={{ duration: 0.8, ease: SARAJOO_EASE, delay }}
        className={className}
      >
        {children}
      </motion.div>
    </div>
  );
}
```

**적용 대상:** 모든 페이지의 **메인 헤드라인**, BODY·MIND·SOUL 순차 등장, 제품 페이지 공감 후킹

### 4-3. 패럴랙스 스크롤 (v4.0 신규)

```typescript
// components/ui/ParallaxImage.tsx
"use client";
import { motion, useScroll, useTransform } from 'framer-motion';
import Image from 'next/image';
import { useRef } from 'react';

export default function ParallaxImage({
  src, alt, className
}: {
  src: string; alt: string; className?: string;
}) {
  const ref = useRef(null);
  const { scrollYProgress } = useScroll({
    target: ref,
    offset: ["start end", "end start"]
  });
  const y = useTransform(scrollYProgress, [0, 1], ["-10%", "10%"]);

  return (
    <div ref={ref} className={`overflow-hidden ${className}`}>
      <motion.div style={{ y }} className="h-[120%] w-full relative">
        <Image src={src} alt={alt} fill className="object-cover" />
      </motion.div>
    </div>
  );
}
```

**적용 대상:** 홈 Hero 배경, 스토리 페이지 이미지, 리추얼 Morning/Night 배경

### 4-4. 레이아웃 시스템 — 비대칭의 미학

#### 정형 그리드 vs 비대칭 그리드

```
v3.0 (정형 2단)                    v4.0 (비대칭 오프셋)
┌──────────┬──────────┐            ┌────────────────┬──────┐
│  이미지   │  텍스트   │            │    이미지       │      │
│  50%     │  50%     │            │    60%         │ 40%  │
│          │          │            │               │텍스트 │
└──────────┴──────────┘            │               │      │
                                   └────────────────┴──────┘
                                        ↕ 다음 블록은 좌우 반전
                                   ┌──────┬────────────────┐
                                   │      │    이미지       │
                                   │ 40%  │    60%         │
                                   │텍스트 │               │
                                   └──────┴────────────────┘
```

**비대칭 원칙:**
- 이미지:텍스트 비율은 **60:40** 또는 **55:45** (이미지가 항상 더 큼)
- 텍스트 블록에 **상단 오프셋** — 이미지 상단보다 약간 아래에서 시작 (`mt-8 md:mt-16`)
- 교차 배치 — 홀수 블록은 이미지 왼쪽, 짝수 블록은 이미지 오른쪽
- 모바일에서는 1열 스택 (이미지 위, 텍스트 아래)

**Tailwind 구현:**
```html
<!-- 비대칭 그리드: 이미지 60%, 텍스트 40% + 상단 오프셋 -->
<div class="grid grid-cols-1 md:grid-cols-[3fr_2fr] gap-8 md:gap-16 items-start">
  <div><!-- 이미지 (60%) --></div>
  <div class="md:mt-16"><!-- 텍스트 (40%, 아래로 오프셋) --></div>
</div>

<!-- 다음 블록: 좌우 반전 -->
<div class="grid grid-cols-1 md:grid-cols-[2fr_3fr] gap-8 md:gap-16 items-start">
  <div class="md:mt-16 md:order-1"><!-- 텍스트 --></div>
  <div class="md:order-2"><!-- 이미지 --></div>
</div>
```

#### 여백 시스템

| 요소 | 모바일 | 데스크톱 | 비고 |
|------|--------|---------|------|
| 섹션 간 여백 | py-20 (80px) | **py-32** (128px) | v3.0 py-24보다 확대 — 호흡감 |
| 블록 내 간격 | gap-8 (32px) | gap-16 (64px) | 이미지↔텍스트 사이 |
| 텍스트 오프셋 | 없음 | mt-16 (64px) | 비대칭 배치용 |
| 컨테이너 | px-6 | max-w-screen-xl + px-8 | 양쪽 패딩 여유 |
| 헤드라인↔본문 | mt-4 | mt-6 | 타이포 간 간격 |

#### 자간·행간·무게감

| 용도 | 서체 | 무게 | 자간 | 행간 | 모바일 | 데스크톱 |

| 영문 서브 헤드라인 | Cormorant Garamond | SemiBold 600 | **0.08em** | 1.2 | text-2xl | text-4xl |

#### 서체 혼용 원칙 — "전통의 현대적 재해석"

| 역할 | 서체 | 느낌 | 적용 |
|------|------|------|------|
| 클래식·우아 | Cormorant Garamond (Serif) | 전통, 격조, 아트 | 영문 헤드라인, 브랜드명, 슬로건 |
| 모던·정갈 | Pretendard (Sans-serif) | 현대, 깔끔, 신뢰 | 한글 본문, 설명, CTA 텍스트 |
| 기능·시스템 | Montserrat (Sans-serif) | 경량, 기능적 | 라벨, 캡션, 카테고리 |

> 세리프(전통) + 산세리프(현대)의 대비가 **"한국 전통 미용법의 현대적 재해석"**이라는 브랜드 스토리를 타이포그래피로 구현한다.

**Tailwind 설정:**
```typescript
// tailwind.config.ts
theme: {
  extend: {
    fontFamily: {
      serif: ['Cormorant Garamond', 'serif'],
      sans: ['Pretendard', 'sans-serif'],
      display: ['Montserrat', 'sans-serif'],
      mono: ['DM Sans', 'sans-serif'],  // 숫자 임팩트용
    },
    letterSpacing: {
      'serif-wide': '0.05em',    // 영문 헤드라인
      'serif-wider': '0.08em',   // 영문 서브 헤드라인
      'label': '0.2em',          // 영문 라벨 (대문자)
    },
    lineHeight: {
      'body-kr': '1.8',          // 한글 본문 행간
      'heading': '1.1',          // 영문 헤드라인
    },
  }
}
```

#### 노이즈 텍스처 (v4.0 신규)

미세한 그레인 텍스처를 배경에 오버레이하여 **디지털 화면에 종이 질감**을 더한다.

```css
/* globals.css */
.noise-overlay::after {
  content: '';
  position: fixed;
  inset: 0;
  opacity: 0.03;
  pointer-events: none;
  z-index: 9999;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)'/%3E%3C/svg%3E");
}
```

- `opacity: 0.03` — 거의 보이지 않지만 무의식적으로 고급 질감을 느끼게 함
- `<body>` 태그에 `noise-overlay` 클래스 적용

### 4-8. Social Proof 비주얼 (v4.0 강화)

#### 숫자 카운터 — Thin Weight + 프로그레스 링

```
    ┌───┐
    │   │ ← 원형 프로그레스 링 (골드 라인, 2px)
    └───┘
   1,250°C       ← DM Sans Thin 100, text-7xl~8xl
   고온 소성       ← Montserrat 대문자, text-xs, 0.2em 자간
```

```typescript
// components/ui/CountUpNumber.tsx
"use client";
import { motion, useInView, useMotionValue, useTransform, animate } from 'framer-motion';
import { useRef, useEffect } from 'react';
import { SARAJOO_EASE } from '@/lib/motion';

export default function CountUpNumber({
  target, suffix = '', label
}: {
  target: number; suffix?: string; label: string;
}) {
  const ref = useRef(null);
  const isInView = useInView(ref, { once: true });
  const count = useMotionValue(0);
  const rounded = useTransform(count, (v) => Math.round(v).toLocaleString());

  useEffect(() => {
    if (isInView) {
      animate(count, target, {
        duration: 2,
        ease: SARAJOO_EASE,
      });
    }
  }, [isInView, target, count]);

  return (
    <div ref={ref} className="text-center">
      <div className="relative inline-flex items-center justify-center w-28 h-28 md:w-36 md:h-36">
        {/* 원형 프로그레스 링 */}
        <svg className="absolute inset-0" viewBox="0 0 100 100">
          <motion.circle
            cx="50" cy="50" r="46"
            fill="none" stroke="#B8986E" strokeWidth="1"
            strokeLinecap="round"
            initial={{ pathLength: 0 }}
            whileInView={{ pathLength: 1 }}
            viewport={{ once: true }}
            transition={{ duration: 2, ease: SARAJOO_EASE }}
          />
        </svg>
        <div className="font-mono font-thin text-5xl md:text-7xl text-brand-dark tracking-wide">
          <motion.span>{rounded}</motion.span>{suffix}
        </div>
      </div>
      <p className="font-display text-xs tracking-label uppercase text-brand-muted mt-3">
        {label}
      </p>
    </div>
  );
}
```

#### 신뢰 배지 — 미니멀 라인 아이콘

